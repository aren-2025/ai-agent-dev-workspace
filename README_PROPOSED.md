# AI Agent Dev Workspace

A documentation-first workspace for independent developers who use coding assistants to plan and review software projects.

This project is early-stage. It is built to be honest, useful, and easy to improve.

## Purpose

The workspace helps a developer move from idea to plan to reviewed change. It provides simple templates and guides for project briefs, product scope, architecture notes, task planning, and human review.

## Audience

- Independent developers
- Small teams
- Open-source maintainers
- Builders using coding assistants

## What this repo includes

- Usage guide
- Architecture notes
- Agent workflow guide
- Planning templates
- Task template
- Example prompt
- Profile README draft
- Publication plan

## Workflow

1. Write the brief.
2. Define the product scope.
3. Draft the architecture.
4. Split work into tasks.
5. Ask a coding assistant to implement one small step.
6. Review the result.
7. Document what changed.
